This folder contains a file named X.sid, which comprises the dataset signals necessary for identifying the transfer function of the PMSM (Permanent Magnet Synchronous Motor) using Matlab.

To execute the contents of this folder, the following are required:

-Matlab R2024 or earlier versions.
-Matlab System Identification Toolbox.
-A foundational understanding of the System Identification Toolbox, which can be acquired through resources like the following tutorial: https://www.mathworks.com/videos/introduction-to-system-identification-toolbox-68901.html

The dataset is taken from [1].


[1]: J. C. Nustes, D. P. Pau, G. Gruosso, "Field oriented control dataset of a 3-phase permanent magnet synchronous motor," Data in Brief, vol.47, 109002, pp. 1-11, 2023
