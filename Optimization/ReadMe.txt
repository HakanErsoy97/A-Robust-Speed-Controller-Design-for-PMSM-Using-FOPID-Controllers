This folder contains a file named 'PMSM_Optimization' designed to optimize PID and FOPID parameters for a specified transfer function

To execute the contents of this folder, the following are required:

-Matlab R2024 or earlier versions.
-Matlab Simulink
-Matlab Simulink Control Design Toolbox.
-A foundational understanding of the Simulink Control Design Toolbox which can be acquired through resources like the following tutorial: https://www.youtube.com/watch?v=lyLffjAwOyQ
